🚀 DigiTools - Modern Product & Cart Web App
A modern React-based digital tools marketplace where users can browse products, add items to cart, and complete checkout with a clean and responsive UI.
This project demonstrates React fundamentals, state management, reusable components, UI design implementation, and user interaction handling.

📌 Project Overview
DigiTools is a responsive web application that allows users to:
•	Browse digital products
•	Add products to cart
•	View cart items dynamically
•	Remove items from cart
•	Checkout products
•	Get real-time notifications
The project focuses on clean UI, component architecture, and React best practices.

🎯 Challenges Faced
1️⃣ Cart State Synchronization
Keeping cart count synced between:
•	Navbar
•	Cart Tab
•	Product Section
Solution:
Centralized cart state inside App.jsx.
________________________________________
2️⃣ Prevent Duplicate Products
Users could add the same product multiple times.
Solution:
Used find() method to check existing items before adding.
________________________________________
3️⃣ UI Matching With Figma Design
Maintaining spacing, layout, and component consistency.
Solution:
Tailwind utility classes + reusable components.
________________________________________
4️⃣ Component Communication
Passing functions across multiple components.
Solution:
Props-based data flow and handler passing.
